Place Clash Royale sound snippets here, named with the card ID, for example:

1.mp3    # Knight
5.mp3    # P.E.K.K.A
7.mp3    # Balloon

Files must be short MP3 or OGG clips. The Sound Quiz will automatically attempt
`/sounds/cards/<cardId>.mp3`. If a file is missing, the game falls back to a
procedural tone so players can continue while you add real audio.
